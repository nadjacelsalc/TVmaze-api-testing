# TVmaze API Testing

Ovo je **TVmaze API testing projekt** razvijen za testiranje javnog **TVmaze REST API‑ja** koristeći Cypress end‑to‑end testove.  
Cilj projekta je validirati REST endpoint‑e, odgovore API‑ja i osigurati očekivano ponašanje kroz automatizirane testove.

API koji se testira: TVmaze Public API — besplatan REST API za TV serije i epizode, dostupan bez autentikacije. :contentReference[oaicite:0]{index=0}

---

## Sadržaj repozitorija

- `cypress/` — Cypress testovi i konfiguracija test slučajeva
- `cypress.config.js` — Konfiguracija test runner‑a
- `lista endpointa i zahtjeva.docx/pdf` — Lista endpoint‑a API‑ja i pripadajućih zahtjeva za testiranje

---

## Svrha projekta

Ovaj projekt je napravljen kako bi:

- Demonstrirala kako pisati automatizirane API testove koristeći Cypress.
- Provjeravala ispravnost REST odgovora za TVmaze API.
- Služio kao referenca za testiranje API‑ja u realnim scenarijima.

---

## Tehnologije

Projekt koristi:

- **Cypress** — za automatizirane testove (API i end‑to‑end)
- **JavaScript** — programski jezik testova
